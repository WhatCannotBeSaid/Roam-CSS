# Roam Research CSS 主题

这是一个自定义的 Roam Research CSS 主题项目。

## 项目结构

- `roam.css` - 主 CSS 文件
- `variables.css` - CSS 变量定义
- `components/` - 组件样式
- `layouts/` - 布局样式

## 使用方法

1. 复制 `roam.css` 的内容
2. 在 Roam Research 中，进入 **All Pages** > **roam/css**
3. 粘贴 CSS 代码到笔记中

## 主要功能

- [ ] 自定义配色方案
- [ ] 优化排版
- [ ] 增强视觉效果
- [ ] 响应式设计
